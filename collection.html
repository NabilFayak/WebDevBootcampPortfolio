Lab 1 Lab 2 Lab 3 Lab 4 Lab 5
This was the very first lab. All we had to do was complete a very simple HTML page. It consisted of a couple tags, mostly just a couple paragraphs of text, very barebones. In this second lab we were able to insert images. Inserting images allowed us to customize the page along with other features. We were also able to use bold and italics. We were able to insert hyperlinks within this lab. Adding links is a very important feature for directing to other websites. You can even use the anchor to hyperlink an image. This lab allowed us to create a table and format it with multiple rows or columns. Within this specific lab I used the rows and columns to outline different shades of blue. This lab marked our very first encounter with CSS. Although this was just in-line formatting, being able to use CSS opened up various customization opportunities.
Lab 6 Lab 7 Lab 8 Lab 9 Lab 10
Using CSS isn't the only way to make a webpage look nice. In this lab I was able to create a favicon, which is the icon that's on the page tab. I also created a page banner to put at the top of the page. This lab was almost like a mini-portfolio. We were able to create an index of all the labs made so far. Putting together all these files for the first time was really exciting. It showed how much we'd done so far. Filling up space is important, it helps to visualize formatting. HTML has a built-in text filler feature that I used, which are the Lorem ipsum words. This feature was used to fill-in a javascript description. Internal CSS works just fine, but being able to use external CSS make the entire project easier to handle. With this external CSS, I made a navigation bar in order to navigate between webpages. This lab was my first taste of Javascript. It served to introduce the many uses for javascript and how to make a site interactive. This lab used javascript to change and alter a piece of text within the body.
Lab 11 Lab 12 Lab 13 Lab 14 Lab 15
Being able to use javascript opens up myriad options for how to make a website. This lab made a fortune telling site that told you your future. It took user inputs and variables to output the user's fortune. This lab used javascript window commands to prompt user inputs. These inputs were then used to complete a quadratic equation or to find the volume of a cone. The user decides which to calculate. Using simple javascript arithmetic, I was able to receive user inputs and store them in variables. These variables were then used to find the user's age in a specific future year the user decides on. This piece of work uses arrays and for loops to calculate the total sum of a shopping cart. This is done by receiving the amount of each item type from the user, and then calculating the data. This was a fun lab that just made a ninja. This ninja does anything it wants to do, as it uses javascript random() functions to do random activities. These involve punching, kicking, karate chopping, etc.
